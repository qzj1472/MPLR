# Installation Notes

- You can choose the installation location in this MSI installer.
- Automatic updates use the current installed MPLR location and update it in place through Velopack.
- If startup on boot is enabled later from the tray menu, MPLR writes a startup command for the current installation path.
- The installer does not enable startup on boot by itself.
- Installing MPLR into another folder manually creates a separate copy.

# 安装说明

- 这个 MSI 安装器支持选择安装位置。
- 软件内自动更新会通过 Velopack 更新当前已安装的 MPLR 目录。
- 如果之后在托盘菜单中开启开机自启动，MPLR 会写入当前安装目录对应的启动命令。
- 安装器本身不会自动开启开机自启动。
- 手动把 MPLR 安装到另一个目录会创建一个独立副本。
