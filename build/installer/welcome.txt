Welcome to MPLR.

This installer supports choosing the installation location. MPLR is Windows-only and uses the selected installation folder for application files, automatic updates, shortcuts, and optional startup entries.

欢迎安装 MPLR。

本安装器支持选择安装位置。MPLR 仅支持 Windows，应用文件、自动更新、快捷方式以及可选开机自启动都会以所选安装目录为准。
