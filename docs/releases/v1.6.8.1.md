# MPLR v1.6.8.1

## 中文

- 更新版本号到 1.6.8.1，并准备 7hz_dev 分支更新。
- 移除卡片栏空白区域的下拉导航菜单，优化卡片栏视觉观感。
- 修复卡片栏四周间隔区域不响应双击刷新、长按和右键操作的问题。
- 缩减底部快捷操作区域高度，使主界面更紧凑。
- 将顶部“转码/编码”入口改为“录屏列表”，点击打开独立窗口，窗口仅保留关闭按钮。
- 设置界面新增“周期规则”，支持全天候与自定义星期/时间段巡检。
- 循环周期说明改为秒/分钟单位，视频分段时间新增秒、分钟、小时单位选择。
- 保存设置新增路径层级选择，支持作者/日期层级、平台/作者/日期层级和自定义占位选项。
- 扩展平台识别与图标兜底，主播头像缺失时显示平台或 MPLR 图标。

## English

- Bumped the version to 1.6.8.1 and prepared the 7hz_dev branch update.
- Removed the room-card area dropdown navigation menu for a cleaner card layout.
- Fixed double-click refresh, long-press, and right-click handling around the room-card margins.
- Reduced the bottom quick-action area height for a denser main window.
- Replaced the top transcode/encoding entry with a Screen Records window that only keeps a close button for now.
- Added a schedule rule setting with all-day and custom weekday/time-window polling modes.
- Updated the cycle-period description to seconds/minutes and added seconds/minutes/hours units for video segment time.
- Added save path-level selection for author/date, platform/author/date, and a reserved custom option.
- Expanded platform recognition and avatar fallback so missing streamer avatars show the platform or MPLR icon.
